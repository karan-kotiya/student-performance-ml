# Student Performance Prediction

## Overview
A machine learning project to predict student performance based on academic and demographic features.

## Tech Stack
- Python, Pandas, NumPy
- scikit-learn
- Jupyter Notebook

## How to run
1. Clone repo
2. Create virtualenv
3. Install: pip install -r requirements.txt
4. Open notebook: jupyter notebook Student_Performance.ipynb
